# 青创网商品数据爬虫（QccCrawler）📦

一个针对青创网（[17qcc.com](https://17qcc.com/)）的实用爬虫项目，通过破解 OB 混淆与 RSA 非对称加解密机制，支持自定义搜索关键词和爬取页码，实现商品数据批量抓取并导出为 CSV 格式，步骤清晰易操作，适配各类使用场景～

## 项目简介 📝

- **目标网站**：青创网（https://www.17qcc.com/）—— 专业电商货源批发平台
- **核心功能**：输入搜索关键词（如 “休闲裤男”）和目标页码（爬取到），自动爬取商品名称、售价、店铺名称、店铺链接等核心数据
- **技术亮点**：动态获取加解密核心参数 qccppm 🔑，分离加密 / 解密模块，内置反爬策略，确保爬取稳定性
- **输出格式**：CSV 文件（可直接用 Excel、WPS 打开，数据结构规范）📊
- **适用场景**：电商数据调研、市场分析、商品价格监控、竞品数据收集等

## 技术栈 🛠️

- 核心语言：Python 3.x（推荐 3.8-3.10 版本，兼容性最佳）
- 依赖库：requests（处理网络请求）、execjs（解析 JavaScript 加解密逻辑）、json（处理接口数据）、csv（数据存储）
- 辅助工具：Node.js（执行 JS 代码必备）、浏览器（获取请求头信息）

## 核心实现逻辑（清晰易懂版）🔍

1. **用户输入**：程序启动后，接收你输入的搜索关键词和想要爬取的页码（支持单页或多页）
2. **获取关键参数**：自动访问青创网首页，提取动态生成的`qccppm`参数（这是加解密的核心，没有它无法爬取数据）
3. **数据加密**：将关键词和页码传入 JS 加密文件，生成加密后的请求数据（网站要求必须加密才能访问接口）
4. **发送请求**：携带加密数据向青创网接口发送请求，获取加密后的响应
5. **数据解密**：用之前获取的`qccppm`参数解密响应数据，得到可读的商品列表
6. **数据存储**：将商品数据写入 CSV 文件，支持多页数据连续追加，不会覆盖已有内容

## 完整使用步骤（一步一操作）🚶

#### 1. 安装 Node.js（必须装！）🌐

- 下载地址：https://nodejs.org/zh-cn/download/（选择 “长期支持版”，兼容性更好）
- 安装步骤：按默认提示下一步即可，无需额外配置
- 验证是否安装成功：
  - 打开 “命令提示符” 或 “终端”
  - 输入命令 `node --version`（若显示 “vxx.x.x” 则安装成功）

#### 2. 安装 Python 依赖库 📦

- 打开 “命令提示符” 或 “终端”

- 输入以下命令，按回车执行（等待 1-3 分钟，自动安装所需依赖）：

  ```bash
  pip install requests execjs
  ```

- 若提示 “pip 不是内部命令”：重新安装 Python 并勾选 “Add Python to PATH”，或百度 “Python pip 环境变量配置”（按系统类型查找教程）

### 第二步：配置爬虫（仅需 1 次）⚙️

1. 下载本项目所有文件，放在同一个文件夹中（如 “桌面 /qcc-crawler”）
2. 获取自己的浏览器请求头：
   - 打开浏览器（推荐 Edge/Chrome），访问青创网（https://www.17qcc.com/）
   - 按 F12 打开 “开发者工具”，点击 “网络”（Network）选项卡
   - 在页面搜索任意关键词（如 “T 恤”），找到名称含 “GetPageList” 的请求（可在搜索框输入 “GetPageList” 快速查找）
   - 点击该请求，找到 “请求头”（Headers），复制其中的 “authorization” 和 “app-guestid” 值
3. 修改爬虫主程序：
   - 用记事本（Windows）或文本编辑（macOS）打开`01_爬取青创网.py`文件
   - 找到`self.headers`中的 “authorization” 和 “app-guestid”，替换为你刚才复制的值（保持引号不变，直接替换引号内的内容）
   - 保存文件（Ctrl+S/Command+S）

### 第三步：运行爬虫（获取数据）🚀

1. 按提示输入信息：
   - 输入搜索关键词（示例：休闲裤男、连衣裙女），按回车
   - 输入想要爬取的页码（示例：输入 “3” 爬取第 3 页；输入 “1-5” 爬取第 1-5 页），按回车
2. 等待爬取完成：
   - 程序会显示 “第 X 页，写入 csv 完成！” 的提示，直到所有页码爬取结束
   - 爬取结果会自动保存到`青创网商品数据(实例).csv`文件中

### 第四步：查看数据 📈

- 打开项目文件夹，找到`青创网商品数据(实例).csv`文件
- 双击直接用 Excel 或 WPS 打开，即可看到整理好的商品数据（商品名、价格、店铺名等）

## 常见问题与解决方法 ❌→✅

| 问题现象                         | 可能原因                   | 解决方法                                                  |
| -------------------------------- | -------------------------- | --------------------------------------------------------- |
| 运行命令提示 “找不到模块”        | 未安装依赖库或安装失败     | 重新执行`pip install requests execjs`命令                 |
| 提示 “无法找到 JS 文件”          | 项目文件未放在同一个文件夹 | 将所有文件移到同一文件夹，重新运行                        |
| 爬取失败，提示 “未提取到 qccppm” | 网络问题或请求头配置错误   | 检查网络，重新获取并替换 “authorization” 和 “app-guestid” |
| CSV 文件乱码                     | 编码不兼容                 | 用 Excel 打开时，选择 “数据→自文本”，编码选择 “UTF-8”     |
| 页码输入后提示 “格式错误”        | 输入格式不符合要求         | 按示例输入（单页输数字，多页输 “X-Y”，如 “1-3”）          |

## 注意事项 ⚠️

1. 反爬提醒：程序内置 1-5 秒随机请求间隔，请勿修改代码缩短间隔，否则可能导致 IP 被网站封禁（封禁后需等待一段时间才能恢复）
2. 数据使用：本项目仅用于技术学习和合法数据研究，请勿用于商业用途或大量爬取数据，遵守网站用户协议
3. 版本更新：若青创网后续更新了网站逻辑，可能导致爬虫失效，届时需更新`02_加密逻辑.js`和`03_解密逻辑.js`文件
